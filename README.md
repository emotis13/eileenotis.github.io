# Eileen M. Otis website

This is a static GitHub Pages site.

## Uploading to GitHub Pages

1. Unzip this folder.
2. Upload all files and folders to the root of the GitHub repository.
3. Make sure GitHub Pages is set to publish from the `main` branch and root folder.
4. Replace the book-cover placeholder by adding the cover image to `assets/` and editing the cover section in `index.html` and `books.html`.

## Main files

- `index.html` - Home page
- `research.html` - Research page
- `books.html` - Books & Projects
- `talks.html` - Talks
- `publications.html` - Selected publications
- `cv.html` - CV download page
- `contact.html` - Contact page
- `styles.css` - Site design
- `assets/eileen-otis-headshot.png` - Headshot
- `OtisCV-2026.pdf` - Downloadable CV
